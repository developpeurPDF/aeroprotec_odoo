MRP Operation Worsheets | MRP Operations Additional Work Sheets | MRP Additional Work Sheets

Your module enables Odoo users to add additional worksheets to MRP operations. When a work order is created, these extra worksheets become available to users.

Installation

- Copy mrp_routing_worksheets module to addons folder

- Install the module normally like other modules

1. Add Additional Worksheets

Manufacturign -> Configuration -> Operations -> Operation -> Additional Work Sheets



Upload Sheet



2. Additional Worksheets on Work Orders

Manufacturign -> Operations -> Work Orders -> Additional Work Instructions
