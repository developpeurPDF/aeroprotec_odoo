Manufacturing Subcontracting With Workorder/Routing Subcontracting Management App

Odoo mrp-manufacture subcontracting process management apps helps to manage subcontract process with bill of material routing process which helps to generate purchase order, picking and return picking from mrp workorder when done subcontracting order. this odoo apps also generate subcontracting process report on manufacturing order report.
