Cost Price for BOM Products Odoo App

Total BOM Cost

This module allows you to view BOM cost Price on Product and Product Template which has BOM and Show Total Cost on BOM Form.


Main Features:

  • Allow users to view BOM total cost price on product variant and product template Form.
  • Users can see BOM Total cost on BOM for a selected product template.
  • Users can see BOM Total cost on BOM for a selected product variant.
  • Allow you to select BOM on product template and variant if they want to see cost for specific BOM on product form. This is for cases where there are multiple BOMs for single product and product variants. So the system will allow you to select BOM on product form and show price for that BOM only.
  • Allow you to view Cost and Total cost for BOM components on BOM form as shown.
