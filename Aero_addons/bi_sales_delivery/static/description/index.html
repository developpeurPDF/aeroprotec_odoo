Sales Order Delivery By Date Odoo Apps

Sales Order Delivery By Date Odoo Apps helps to manage expected delivery date of product on sale order line. This delivery date will be also set on stock move/operation, While confirm sales order it will also generate different Delivery Order and manufacturing order based on delivery date, customer lead time and manufacturing lead time.

Features

Allow to add expected Delivery dates.

Allow user to add different expected delivery date for the Sale Order line.

Group By Delivery Order.

Generate multiple Delivery order Based Delivery Date Set on Sale Order line.
Demo Request Video Tutorial

Allow/Disallow Delivery Date in Sale Order Line

Goto Sales -> Configuration -> Settings in that user have access to enable/disable "Allow Delivery Date in Sale Order Line" option to use different delivery date for every sale order line.

Different Delivery Order based on Delivery Dates and Customer Lead Time

User can see different delivery order in that delivery date should be calculated based on delivery date + customer lead time.

Different Manufacturing Order based on Delivery Dates, Customer Lead Time and Manufacturing Lead Time

User can see different manufacturing order in that scheduled date should be calculated based on delivery date + customer lead time - manufacturing lead time.
