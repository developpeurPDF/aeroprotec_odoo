Odoo Metabase Connector

This is our metabase module where we can integrate Metabase, a data analytics visualisation tool, with Odoo.

Here, we can Allow users (with necessary permissions) to configure the API key and Metabase URL in Odoo Settings.

Here, we can fetch all dashboards from Metabase using Metabase API & URL .And Display the dashboards in Odoo in a dedicated menu.A valid Metabase API key will be required to authenticate the connection between Odoo and Metabase.

Here, we can see how to add menu in odoo metabase.Create a new menu (e.g., "Metabase Dashboards") in the Odoo main menu or under the "Reports" section where users can click to view Metabase dashboards.The integration will allow users to fetch all dashboards created in Metabase and display them inside Odoo through a custom menu .

Design the embedded dashboard layout in Odoo to be user-friendly. It should be responsive and adjust to the screen size of the Odoo interface. Optionally cache the list of dashboards in Odoo for performance optimization, updating the cache at a set interval.
